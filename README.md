## Installation
Install Tampermonkey, then click [this link](https://github.com/evan18c/Ghost-Cycler/raw/refs/heads/master/ghost_cycler.user.js).

## Features
The script has the following features:
- Ghost Switching
- Auto Save
- Instant Refresh (CTLR + R)
- Moves Pop Ups
